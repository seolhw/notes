常见正则表达式


- 验证内容中是否含有中文
```
  /[\u4e00-\u9fa5]/ // 20年前的正则表达式，不过还可以用

  /\p{Unified_Ideograph}/u // 最新的正则表达式，目前只有chrome支持

  // 优雅降级为es5

  /[\u3400-\u4DB5\u4E00-\u9FEF\uFA0E\uFA0F\uFA11\uFA13\uFA14\uFA1F\uFA21\uFA23\uFA24\uFA27-\uFA29\u{20000}-\u{2A6D6}\u{2A700}-\u{2B734}\u{2B740}-\u{2B81D}\u{2B820}-\u{2CEA1}\u{2CEB0}-\u{2EBE0}]/u


仅支持中英文及数字  /^[\u4e00-\u9fa5a-zA-Z0-9]+$/
两位小数金额   /^[0-9]+(\.[0-9]{0,2})?$/

仅支持数字、特殊字符    /^[0-9`~!@#$%^&*()_\-+=<>?:"{}|,./;'\\[\]·~！@#￥%……&*（）——\-+={}|《》？：“”【】、；‘'，。、]+$/

仅限中文、字母，首尾无空格  /^[a-zA-Z0-9\u4e00-\u9fa5]+[a-zA-Z0-9\u4e00-\u9fa5\s]*[a-zA-Z0-9\u4e00-\u9fa5]+$/


校验身份证号: /^(\d{18,18}|\d{15,15}|\d{17,17}[x,X])$/

校验手机号：/^(13[0-9]|14[579]|15[0-3,5-9]|16[67]|17[0135678]|18[0-9]|19[189])\d{8}$/

```
