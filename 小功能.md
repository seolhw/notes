# 每行显示21个字符，每页显示4行

~~~javascript

var info = '系统公告：体检云本期更新内容：增加前台登记、收银及退款功能，支持导检单&条码打印，支持延期补检，支持前台叫号机、智能储物柜使用；支持科室排队系统、科室加项功能使用。科室加项功能使用'
function log(info) {
  let pageList = []
  let rowList = []
  let fontString = ''
  for (let i = 0; i < info.length; i++) {
    fontString += String(info[i])
    if (fontString.length === 21) {
      rowList.push(fontString)
      fontString = ''
    }
    if (rowList.length === 4) {
      pageList.push(rowList)
      rowList = []
    }
  }
  if (fontString.length) {
    rowList.push(fontString)
  }
  if (rowList.length) {
    pageList.push(rowList)
  }
  return pageList
}
const list = log(info)
console.log(list)
~~~

思路：先声明三个变量（总的数组，每页的数组，每行的内容），然后循环该字符串，（1）先拼接每行的内容，如果行的字符串的长度达到了21，就把它push进每页的数组，并把行的字符串置为空；（2）然后如果每页的数组的长度达到了4，就把它push进总的数组，并把每页的数组置为空数组；（3）最后循坏结束后，如果该最后一行的内容没有21个字符串，也要push进每页的数组，然后把每页的数组push进总的数组。

# 冒泡排序

~~~javascript
let arr1 = [8, 6, 3, 3, 4, 7, 9]
function sort(arr1) {
  let  arr=arr1
  for (let i = 0; i < arr.length; i++) {
    for (let j = 0; j < arr.length - i; j++) {
      if (arr[j] < arr[j + 1]) {
        let a1 = arr[j + 1]
        arr[j + 1] = arr[j]
        arr[j] = a1
      }
    }
  }
  return arr
}
sort(arr)
~~~

思路：双重循环，第一层循环整个数组，第二层循环整个数组减去循坏的次数（因为每次循坏已经把最大或者最小的数，放在了最后一位），比较相邻的两位，根据条件互换位置。

# 小球沿抛物线运动

~~~javascript
<template>
  <div class="throwBox">
    <button @click="run">水平400px/s</button>
    <div class="range">
      <div class="box"></div>
    </div>
  </div>
</template>
<script>
export default {
  data () {
    return {}
  },
  methods:{
    /*
    * x:水平的距离
    * y:垂直的距离
    * t:时间间隔
    * n:执行次数
     */
    run(){
     let x,y,t=0.1,n=0
     let dom=document.getElementsByClassName('box')[0]
     let timer=setInterval(()=>{
      n++
      x=400*t*n //x轴移动的距离
      y=0.5*980*Math.pow(n*t,2) //y轴移动的距离 --- 抛物线式的运动轨迹
      // y=0.001*Math.pow(x,2) //y轴移动的距离 ——— 二次函数式的运动轨迹  y=ax²+bx+c 二次项系数a的绝对值越大开口越小
      dom.style.left=x+'px'
      dom.style.top=y+'px'
      // dom.style.setProperty('transform',`translate(${x}px,${y}px)`)
      if(x>=480 || y>=480){ //用x,y的边界来判断
        clearInterval(timer)
      }
    },100)
    }
}
}
</script>
<style lang="scss" scoped>
.range{
  width: 500px;
  height: 500px;
  border:1px solid black;
}
.box{
  width: 20px;
  height: 20px;
  background:red;
  border-radius: 50%;
  position: relative;
}
</style>
~~~

思路：先确定好小球的运动轨迹（如：抛物线、二次函数等），然后利用自身定位或者transform加translate的方式来确定小球的x轴和y轴的运动距离即可。
